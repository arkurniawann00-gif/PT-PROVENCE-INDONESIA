# Document Management System - PT PROVENCE INDONESIA

Proyek ini adalah sistem pengelolaan dokumen digital yang dirancang untuk meningkatkan efisiensi administrasi di PT PROVENCE INDONESIA.

## Fitur Utama
* Upload dan penyimpanan dokumen terpusat.
* Pencarian dokumen cepat.
* Manajemen hak akses pengguna.

## Cara Instalasi
1. Clone repositori ini.
2. Jalankan perintah `npm install` (atau sesuai bahasa pemrogramanmu).
3. Konfigurasi file `.env`.
